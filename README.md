# Light and Memory Audio — Website

This repository hosts the Light and Memory Audio website via **GitHub Pages**.

## Where the website files live
The live site is served from the **/docs** folder:

- `docs/index.html` — the website
- `docs/assets/` — images (logo, etc.)
- `docs/.nojekyll` — tells GitHub Pages not to run Jekyll

## Updating the website
1. Open `docs/index.html`
2. Edit and commit changes
3. Refresh the live site after 1–2 minutes (GitHub Pages deploy time)

## Live site
Add your live link here once you have it working:
- https://YOURUSERNAME.github.io/REPO/

## Notes
- Keep filenames lowercase (example: `lama.jpeg`)
- GitHub Pages is case-sensitive (so `lama.jpeg` ≠ `Lama.jpeg`)
